qurl.sh

Fast, terminal friendly ephemeral file sharing.

The fastest way to pipe files to the web.

Upload

Upload a file from your terminal.

$ curl -T file.txt https://qurl.sh

With Password:

$ curl -T file.txt -H "X-Password: 123" https://qurl.sh

TTL & Limits: (Max 7d and 100 downloads)

$ curl -T file.txt -H "X-TTL: 1h" https://qurl.sh
$ curl -T file.txt -H "X-Downloads: 5" https://qurl.sh

Download

Get your file back with a simple command.

$ wget https://qurl.sh/abcd/file.txt
$ curl -O https://qurl.sh/abcd/file.txt

View QR:

$ open https://qurl.sh/qr/abcd/file.txt

Pretty Print

Visualize JSON, YAML, or XML.

$ curl -F "file=@c.yaml" https://qurl.sh/pretty

Encrypted Secrets

Share text securely (AES-128).

$ echo "secret" | curl -d @- https://qurl.sh/secret

Code Viewer

View code with syntax highlighting.

Just open the file link in your browser.

Default: Files auto-delete after 1 download. Customize with X-TTL or X-Downloads. Max 50 MB.
