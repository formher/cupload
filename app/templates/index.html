qurl.sh

Fast, terminal friendly ephemeral file sharing.

The fastest way to pipe files to the web.

Upload

Upload a file from your terminal.

$ curl -T file.txt https://qurl.sh

With Password:

$ curl -T file.txt -H "X-Password: 123" https://qurl.sh

Download

Get your file back with a simple command.

$ wget https://qurl.sh/abcd/file.txt
$ curl -O https://qurl.sh/abcd/file.txt

View QR:

$ open https://qurl.sh/qr/abcd/file.txt

Pretty Print

Visualize JSON, YAML, or XML configs.

$ curl -F "file=@c.yaml" https://qurl.sh/pretty

Files auto-delete after the first successful download. Max upload size: 50 MB.
