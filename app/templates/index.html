cupload.io

Fast, ephemeral file sharing for developers

Upload

Upload a file from your terminal.

$ curl -T file.txt https://cupload.io

# With Password
$ curl -T file.txt -H "X-Password: 123" https://cupload.io

Pretty Print

Visualize JSON, YAML, or XML configs.

$ curl -F "file=@c.yaml" https://cupload.io/pretty

Files auto-delete after the first successful download. Max upload size: 50 MB.
